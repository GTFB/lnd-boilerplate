# Quick Start Guide

This guide will help you get started with LND Boilerplate quickly.

## Prerequisites

Before you begin, make sure you have:

- Node.js 18+ installed
- Bun package manager
- Git

## Installation

1. Clone the repository:
```bash
git clone https://github.com/your-repo/lnd-boilerplate.git
cd lnd-boilerplate
```

2. Install dependencies:
```bash
bun install
```

3. Start the development server:
```bash
bun run dev
```
